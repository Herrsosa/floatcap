# Floatcap Agent Router

Read the following routing rules carefully before taking any action.

- **If modifying HTML files (`*.html`)**: Read `.agents/rules/html-rules.md`
- **If modifying styles (`src/style.css`)**: Read `.agents/rules/css-rules.md`
- **If modifying or creating JavaScript modules (`src/*.js`)**: Read `.agents/rules/js-rules.md`
- **If adding a new blog article**: Read `.agents/skills/create-article.md`
